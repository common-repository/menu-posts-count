Display total number of posts(or custom post type) count of a category, tag or custom taxonomy just after the menu item in navigation bar. 

== Description ==

Menu Posts Count display total count of posts, product or custom post type in navigation menu supported by your theme. It is simple to use, just install the plugin from WordPress plugin repository, activate it, go to the setting page and select the menu and taxonomy type for which you want to show total counts. It has several settings to control the counts display and styling. 

**Available settings**

* Show counts in first level menu - Allow you to show counts in first level menu
* Show only for count greater than - You can set the limit below which counts will be not shown.
* Which menus - Select available menus from your theme for which you want to show counts.
* Select taxonomies - Select taxonomies for which you want to count the post and show it in menu.
* Select bracket type - You can select parentheses or brackets to enclose the counts.
* Keep space around the counts - You can put space around the counts inside the brackets.

To report a bug or give suggestions, feel free to email me at sanjay.ojha@mail.com.    

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/menu-posts-count` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Use the "Menu Posts Count" menu to configure the plugin.

== Frequently Asked Questions ==

= Does it support custom taxonomy? =

Yes, it shows all taxonomies which is available in menu and front end when you choose settings from admin area. By default it show category and tag.

= How can I style counts? =

Counts are put inside `<span>` HTML tag with class `mpc-count`. You can use this class to write css in your theme style files to get desired styling.
